/3.1_Glycine_max_JGI_Wm82.a4.v1.cleaned.gaf.gz
  Final result: functional annotation set for the protein coding genes in the
  Joint Genome Institute (JGI) Glycine Max genome assembly Wm82.a4.v1 (genotype
  Williams 82, assembly 4.0, gene model annotation 1.0). These are the
  annotations we will submit to the GO databases.

/3.2_locus_mapping_to_a2.txt
  Mapping of the gene identifiers of this assembly to the Wm82.a2.v1 assembly.
  This file was downloaded as-is and renamed in April 2019 from 
   https://legumeinfo.org/data/public/Glycine_max/Wm82.gnm4.ann1.T8TQ/glyma.Wm82.gnm4.ann1.T8TQ.info_locus_transcript_name_map.txt.gz
